## ▶️ How to Run the Projects

### 1. Clone the repository

```bash
git clone https://github.com/chandan0149/olyms-mern-daily-assignments
```

### 2. Navigate to a specific project

```bash
cd "Day 10 - Practice Assignment"
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the project

```bash
npm start
```
